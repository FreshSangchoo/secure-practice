# AWS 기초 강의 - 실습 중심 학습 사이트

이커머스 서비스(React + Node.js 쇼핑몰)를 AWS에 배포하면서 배우는 **AWS 입문 과정**입니다.
(현대오토에버 모빌리티 SW 스쿨 IT 보안 내용)
